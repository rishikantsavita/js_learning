what is scope

Scope determines the accessibility (visibility) of variables. javaScript variables have 3 types of scope: 1. Block scope 2. Function scope 3. Global scope

What is Variable Scope ?

Variable scope determines where a variable can be accessed or used in your code. In JavaScript, there are three main types of scope: 1. Global Scope , 2. Function (Local) Scope , 3. Block Scope (let and const) , 4. Lexical Scope (Closures)

Quick Summary
Scope Type -- Can be accessed where? -- Declared using
Global Scope -- Anywhere in the script -- var, let, const
Function Scope -- Only inside the function -- var, let, const
Block Scope -- Only inside {} (if, for, while, etc.) -- let, const (NOT var)
Lexical Scope -- Inner functions can access outer variables -- Function closures

What is a Data Type in JavaScript?

Data type refers to the type of data that a JavaScript variable can hold. There are seven primitive data types in JavaScript (Number, BigInt, String, Boolean, Null, Undefined and Symbol). Objects are non-primitives.

1.Primitive Data Types (Immutable & stored by value)

Data Type -- Example -- Description
Number -- 42, 3.14, -7 -- Any number (integer or floating-point)
String -- "Hello", 'World' -- A sequence of characters
Boolean -- true, false -- Represents logical values Booleans are commonly used in conditional statements (if, else, while, etc.) to determine whether a block of code should execute.
Undefined -- let x; -- A variable declared but not assigned a value
Null -- let y = null; -- Represents "nothing" or an empty value
Symbol (ES6) -- Symbol('id') -- Unique, immutable identifier
BigInt (ES11) -- BigInt(1234567890123456789n) -- Large integers beyond Number limits

2. Non-Primitive (Reference) Data Types

Non-primitive data types store complex values and are stored by reference in memory.
Data Type - Example - Description
Object - {name: "John", age: 30} - Collection of key-value pairs
Array - [1, 2, 3, 4] - Ordered list of values
Function - function greet() {} - A block of reusable code

Key Differences: Primitive vs. Non-Primitive

Feature - Primitive Data Type - Non-Primitive Data Type
Storage - Stored by value - Stored by reference
Mutability - Immutable (cannot be changed) - Mutable (can be changed)
Example - let x = 5; - let obj = {name: "Alice"};
Copying - Creates a separate copy - Copies only the reference

Object in JavaScript

In JavaScript, an object is a collection of key-value pairs, where keys are strings (or Symbol) and values can be of any data type, including other objects, arrays, and functions.
